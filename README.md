# 🎮 Emo Games

Welcome to **Emo Games** — a beautifully crafted, self-contained mini-arcade built right into your browser!

**[Play the Live Demo on Vercel!](https://emo-games.vercel.app)**

## 🕹️ About the Arcade
Emo Games is designed to look and feel like a digital arcade room. It features a glowing neon marquee, a modular cartridge system, and a slick full-screen cabinet modal that loads whatever game you select without navigating away from the lobby.

## 👾 The Games Library
The arcade comes racked up with 7 different games across various genres:

1. **Subway Runner** (🛹) - A custom Unity WebGL build of the classic 3D endless runner. Dodge trains, switch lanes, and run forever!
2. **Stack** (🏗️) - A custom 3D stacking game built with Three.js and Cannon.js. Drop blocks precisely and build the tallest tower possible without overhanging.
3. **Tic-Tac-Toe** (❌⭕) - The classic strategy game. Outsmart the seat next to you!
4. **Memory Match** (🧠) - Flip pairs of emojis, build a streak, and beat your move count.
5. **2048** (🔢) - Slide, merge, repeat. Chase the elusive 2048 tile.
6. **Snake** (🐍) - Eat, grow, and don't double back on yourself in this canvas classic.
7. **Bug Pop** (🐛) - 30 seconds on the clock. Tap the bug before it ducks away!

## 🚀 How to Run Locally
Since the entire arcade runs entirely on the client side:
1. Clone the repository.
2. Open `index.html` in your favorite web browser.
*(Or serve it using a local HTTP server like `npx serve` or `python -m http.server`)*

## 🛠️ Technologies Used
- **Arcade Lobby & UI:** Pure Vanilla HTML, CSS (Custom Variables, Flexbox, CSS Grid), and JavaScript.
- **Embedded Games:** HTML5 `<canvas>`, Vanilla JS.
- **Subway Runner:** Unity WebGL.
- **Stack:** Three.js and Cannon.js physics engine.
- **Hosting:** Deployed via Vercel.
