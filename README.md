# 🎮 Emo Games (Ad-Free Poki Clone)

Welcome to **Emo Games** — a beautifully crafted, self-contained mini-arcade built right into your browser!

**[Play the Live Demo on Vercel!](https://emo-games.vercel.app)**

## 🕹️ About the Platform
Emo Games was completely redesigned from the ground up to serve as a premium, 100% ad-free clone of Poki.com. It features a sleek masonry grid UI, dynamic squircle tiles with hover animations, a category sidebar, and a seamless full-screen Player Mode.

## 🚀 How to Run Locally
Since the entire arcade runs entirely on the client side:
1. Clone the repository.
2. Open `index.html` in your favorite web browser.

## 🛠️ Technologies Used
- **UI & Layout:** Pure HTML and Vanilla CSS (Masonry Grids, Flexbox, Variables).
- **Embedded Games:** HTML5 `<canvas>`, Vanilla JS.
- **Hosting:** Deployed via Vercel.
